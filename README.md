SportTracker


Трекер спортрезультатів


Консольна програма для збереження та аналізу спортивних результатів.


Можливості програми:




Додавання спортивного результату.


Перегляд усіх результатів.


Перегляд динаміки результатів.


Пошук найкращого результату.


Видалення записів.


Збереження даних у файл JSON.




Запуск програми


python main.py



Файли проєкту




main.py — основний код програми.


README.md — опис проєкту.
